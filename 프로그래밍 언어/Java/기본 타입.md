자바에는 정수, 실수, 문자, 불리언 4가지의 기본 타입이 있다.

## 정수
정수는 int, long 두 가지고, 각각 4 bytes, 8 bytes 크기를 갖는다. long은 l 또는 L을 붙여야한다.

```java
int small = 10;
long big = 100l;
```

## 실수
실수는 float, double 두 가지고, 각각 4 bytes, 8 bytes 크기를 갖는다. float는 f 또는 F를 붙여야한다.

```java
float small = 12.5f;
double big = 25.5;
```

## 문자
문자는 char 한 가지고, 2 bytes 크기를 갖는다. 작은 따옴표로 리터럴을 표현한다.

```java
char c = 'c';
```

## 불리언
불리언은 boolean 한 가지고, 1 byte 크기를 갖는다. true 또는 false가 할당된다.

```java
boolean isTrue = false;
```
